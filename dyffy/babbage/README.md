## Babbage

The "dyfference" engine.  So modular it hurts.

### Requirements

- Python 2.7 
- PostgreSQL

### Installation

    $ git clone git@github.com:tensorjack/babbage.git

    $ cd babbage

    $ pip install -r requirements.txt

### Unit tests

We love tests.  Our motto is, "Test the living shit out of everything."

    $ nosetests
